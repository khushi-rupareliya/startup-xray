Startup X-Ray

Startup X-Ray is an AI-powered web application that evaluates startup ideas and predicts their success potential through a structured multi-step analysis.

---

Features

* Step-by-step startup evaluation
* AI-based idea analysis
* Market & business assessment
* Execution and financial evaluation
* Final success probability dashboard
* Clean SaaS-style UI (Dark + Light mode)

---

Tech Stack

Frontend

* React.js
* CSS (Custom UI)

**Backend**

* Flask (Python)
* OpenAI API

---

Setup Instructions

1. Clone Repository

```bash
git clone https://github.com/khushi-rupareliya/startup-xray.git
cd startup-xray
```

2. Backend Setup

```bash
cd backend
pip install -r requirements.txt
```

Create `.env` file:

```
OPENAI_API_KEY=your_api_key_here
```

Run backend:

```bash
python app.py
```

---

3. Frontend Setup

```bash
cd frontend
npm install
npm start
```
